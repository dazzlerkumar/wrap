# 2025 Career Wrap - Deepak Kumar

A polished, Instagram-style stories player and stats page for Deepak Kumar's 2025 career achievements.

## Tech Stack
- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS 4**
- **Lucide React**

## Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.

### Static Export
```bash
npm run build
```
The build process generates a static `out` directory which can be hosted on any static web server.

## Features
- **Instagram-like Stories**: Auto-advancing logic with progress bars.
- **Responsive Design**: 9:16 aspect ratio enforced with mobile optimization.
- **Interactive**: Tap Zones, Keyboard navigation, and Pause/Resume.
- **Stats Dashboard**: Comprehensive overview of projects and technical growth.
